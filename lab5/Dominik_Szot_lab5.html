def create_matrix(pow: int):
    matrix = np.zeros((9, pow+1))
    for i in range(len(years)):
        for j in range(pow+1):
            matrix[i][j] = (years[i] ** j)

    return matrix

In [ ]:
def horner(L, x):
    i = len(L) - 1
    result = L[i]
    while i > 0:
        i = i - 1
        result = result*x + L[i]

    return result

Wielomiany Czebyszewa mona zdefiniowa za pomoc funkcji rekurencyjnej:
Tn(x) = 2xTn-1(x) - Tn-2(x)

Dla wielomianu stopnia 2 funkcje bazowe bd postaci:
T0(x) = 1
T1(x) = x
T2(x) = 2x2-1
